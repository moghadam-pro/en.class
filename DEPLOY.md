# Free Discussion Class — Deployment Guide

---

## First Deploy

Follow `INSTALL.md` fully for the initial setup.

---

## Updating the Application (zero-downtime)

```bash
cd /var/www/free-discussion-class

# 1. Pull latest code (or rsync)
git pull origin main
# OR
rsync -avz --exclude='.env' ./free-discussion-class/ user@server:/var/www/free-discussion-class/

# 2. Install/update PHP dependencies
composer install --no-dev --optimize-autoloader

# 3. Rebuild the topic index
php scripts/rebuild-index.php

# 4. Reload PHP-FPM (no downtime)
sudo systemctl reload php8.3-fpm

# 5. Test Nginx config and reload
sudo nginx -t && sudo systemctl reload nginx
```

---

## Deploying New Topics Manually

Place JSON files directly in `data/topics/` then rebuild:

```bash
cp my-new-topic.json /var/www/free-discussion-class/data/topics/
sudo chown www-data:www-data /var/www/free-discussion-class/data/topics/my-new-topic.json
cd /var/www/free-discussion-class
php scripts/rebuild-index.php
```

---

## Via Telegram Bot

1. Open Telegram and message your bot.
2. Send a DOCX or PDF file directly.
3. The bot will parse, save, and reply with the published URL automatically.

Only user IDs listed in `TELEGRAM_ADMIN_IDS` may upload content.

---

## Rollback

If something breaks after an update:

```bash
cd /var/www/free-discussion-class
git log --oneline -10     # find the last good commit hash
git checkout <hash>
composer install --no-dev --optimize-autoloader
sudo systemctl reload php8.3-fpm
```

---

## Environment Changes

After editing `.env`:

```bash
sudo systemctl reload php8.3-fpm
```

PHP-FPM caches environment variables — a reload is required.

---

## Log Rotation

Add to `/etc/logrotate.d/free-discussion-class`:

```
/var/www/free-discussion-class/logs/*.log {
    daily
    missingok
    rotate 14
    compress
    delaycompress
    notifempty
    create 640 www-data www-data
    sharedscripts
    postrotate
        systemctl reload php8.3-fpm > /dev/null 2>&1 || true
    endscript
}
```

---

## Backup

```bash
# Back up all topic data
tar -czf backup-$(date +%Y%m%d).tar.gz \
    /var/www/free-discussion-class/data \
    /var/www/free-discussion-class/.env

# Restore
tar -xzf backup-20260611.tar.gz -C /
php /var/www/free-discussion-class/scripts/rebuild-index.php
```
